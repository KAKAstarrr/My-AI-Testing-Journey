# My-AI-Testing-Journey
#  AI赋能的软件测试自学与实践项目

> 一名计算机科学专业的毕业生，通过系统性的项目实践，从零构建软件测试技能，并探索AI在质量保障中的前沿应用。

## 项目目标
1.  **核心技能构建**：掌握软件测试核心理论与自动化测试工具，达到初级测试工程师的岗位要求。
2.  **AI技术融合**：探索并实践至少一种AI工具（如视觉测试、测试用例生成）在现代测试流程中的应用。
3.  **求职成果导向**：以本项目作为能力证明，成功获得软件测试相关岗位的面试机会与录用通知。

## 学习与实践路线图

### 阶段一：基础夯实 (第1周)
- [x] 搭建学习环境，注册GitHub并创建本项目仓库。
- [x] 学习测试理论基础：测试生命周期、测试用例设计方法（等价类、边界值）。
- [ ] **实践输出**：为选定的Web应用（如豆瓣电影）编写手工测试用例文档。
- [ ] 学习Git基础，用于版本控制与代码托管。

### 阶段二：自动化入门与AI探索 (第2-3周)
- [ ] 学习Python + Selenium/Playwright，实现Web UI自动化测试。
- [ ] **实践输出**：完成第一个自动化测试脚本（例如：网站登录流程）。
- [ ] 研究AI测试工具（如Test.ai, Applitools），了解其应用场景。
- [ ] **实践输出**：在自动化脚本中集成一个AI测试步骤（如视觉验证）。
- [ ] 学习接口测试工具Postman。

### 阶段三：集成与求职准备 (第4周)
- [ ] 将自动化测试集成到简易的CI流程（使用GitHub Actions）。
- [ ] **实践输出**：整理项目所有产出，撰写一篇技术总结博客。
- [ ] 优化简历，开始针对性投递。

- Postman | Git | Markdown
